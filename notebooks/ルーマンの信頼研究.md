# ルーマンの信頼研究

## 要約 (NotebookLM AI要約)

ニクラス・ルーマンの社会学理論のうち、**複雑性管理**・**信頼**・**権力**を扱う。
複雑性管理とは、不確実性と非線形なダイナミクスを乗り越えるための手法であり、
組織は外部環境を映し出すためにある程度の複雑性を内部に取り込む必要がある。
ルーマンは、信頼と権力を「**一般化されたコミュニケーション・メディア**」として捉え、
それらが未来の膨大な可能性を管理可能な確実性へと縮減することで社会的相互作用を
単純化すると論じる。伝統的な社会学が道徳的・因果的説明に依拠しがちなのに対し、
ルーマンのシステム理論は、意味生成のプロセスがいかに社会を時間的に安定させるかに
焦点を当てる。これらのテキストは、社会システムがいかに境界を維持し、予測不可能で
相互接続の進む世界で存続するための内的論理を作り出すかを示している。
