# シラバス：異文化コミュニケーション入門

**授業概要**
さまざまな文化的背景を持つ学生が集まるAPUの多文化キャンパスでは、学生がお互いへの関心・好奇心を持ちながら、コミュニケーションを通じて理解し合おうとすること、学生間で学び合う関係性を育むことが個人の成長の鍵となる。この授業では、異文化間コミュニケーションの基本的概念を理解し、実践での応用力を高めることを狙いとする。

**到達目標**
1. 自らの異文化体験を意識的に振り返り、考察することができる。
2. 自分自身のコミュニケーションスタイルに気づき、状況に応じて工夫できる。
3. 自分自身の文化的価値観を理解し、説明することができる。
4. 学生のさまざまな文化的背景に意識を向け、お互いの理解を深めることができる。

**毎回の授業概要（全14週）**
第1週 導入 / 第2週 経験学習モデル / 第3週 非言語コミュニケーション / 第4-5週 言語コミュニケーション / 第6週 異文化感受性 / 第7週 文化と価値観 / 第8週 多様なアイデンティティ / 第9週 ステレオタイプ / 第10週 グループプロジェクト(1) / 第11週 異文化環境適応 / 第12週 グループプロジェクト(2) / 第13週 発表 / 第14週 まとめ

**成績評価**
レポート試験30%（最終個人レポート）、発表・プレゼン50%（中間・最終グループプロジェクト+ピア評価）、日常評価20%（参加・振り返りジャーナル）

**AIポリシー**
限定使用：ブレインストーミング、構成作成、修正への活用可。生成AIコンテンツは適切なクレジット・引用が必要。無許可使用はアカデミック・インテグリティ違反。
