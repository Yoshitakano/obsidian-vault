# サービスマネジメント入門2.3

**品質・マーケティング・生産性**の交点に焦点を当てた、サービスマネジメントの包括的な枠組みを提供するテキスト。**マクドナルドとモスバーガー**という業界大手の詳細な分析を通じ、**主観的な顧客期待**と、期待と実際の体験との**心理的ギャップ**がサービス価値をどう定義するかを解説する。**SERVQUALモデル**や**サービス・プロフィット・チェーン**といった構造的ツールを体系的に検証し、社内業務と従業員エンゲージメントが**顧客ロイヤルティと企業収益性**を直接どう牽引するかを示す。理論から実践へと移行し、**戦略的価格設定手法**と、人的要素を損なわずに**サービス生産性**を維持する独特の課題を検討する。
