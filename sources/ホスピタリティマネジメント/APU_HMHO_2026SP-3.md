# APU_HMHO_2026SP-3

APUの講義資料。**顧客価値**の概念を中心に、ホスピタリティ・マーケティングとオペレーションの基本原理を紹介する。サービスの**共創的な性質**(提供者の提供内容と、顧客が既存の規範・システムに従う能力の両方に成功が依存する)を探る。「扱いやすい顧客」と「扱いにくい顧客」の区別を分析し、企業が顧客層を厳しく選別すべきか、難しい要求を成長・収益の機会として受け入れるべきかを学生に考えさせる。最終的に**顧客価値**を、知覚された効用・便益と消費者が負担する総コストとのバランスとして定義する。
