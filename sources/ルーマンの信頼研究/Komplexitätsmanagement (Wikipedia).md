# Komplexitätsmanagement (Wikipedia)

複雑性管理を、現代組織の予測不可能で非線形なダイナミクスを乗り越えるための戦略的アプローチとして
解説する。内的複雑性と外的複雑性を区別し、システムは「準客体(quasi-objects)」やシミュレーションを
通じて環境を選択的に再構築することで安定を保つべきだと論じる。社会的信頼とミクロ政治が、
システム内の緊張を生み出すと同時に縮減する主要な駆動力であるとする人間的要素の役割も中心的に扱う。
製造業やバリューチェーンにおける実践的応用にも触れ、市場主導の多様性とコスト効率の良い内部プロセスの
バランスを取ることの重要性を強調している。
