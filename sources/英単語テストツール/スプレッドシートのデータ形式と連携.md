# スプレッドシートのデータ形式と連携

Googleスプレッドシートをデータベースとして利用する、Webベースの英単語学習アプリケーションの
技術サポート記録。スプレッドシートは「カテゴリ・フォルダ・単語ペア」で構成される特定の
データアーキテクチャに従う必要があり、**英単語そのものが一意の識別子**としてユーザーの学習進捗を
追跡する役割を果たす。アプリを正しく動作させるためには、シートを**CSVとして公開**する際の
厳密な手順を守り、localStorageの履歴とのリンクが壊れないようデータの整合性を保つ必要がある。
さらに、誤答の**色分け表示**や効率的な学習のための**復習リストの自動生成**といった主要機能を
復旧するためのデバッグ済みコードも提供している。
