# 言語哲学班_本番

ウィトゲンシュタイン『論理哲学論考』のうち「論理空間」と「可能性」の関係に焦点を当てた本番用
発表資料。「世界」を確立された事実の集合と定義し、「論理空間」は事実として真であるか否かに
関わらず理論上考えうるあらゆる事態を包含するとする。ある文が有効な「命題」となるのは、関わる
「対象」がその行為を可能にする「実体」を共有している場合であり、例えば空を飛ぶ犬は抽象化を
通じて論理的に可能だが、火山のように噴火する猫は不可能だと説明する。対象の「実体」がその
事態を内在的に支えうるかを分析することで、ある思考が論理空間内に存在するかを判定する枠組みを
示す。
