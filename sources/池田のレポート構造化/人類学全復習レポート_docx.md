# 人類学全復習レポート(docx版)

文化人類学講義の全体を復習する一連のレポート。自らのフィールドワークへの動機から始まり、
「驚異の部屋(キャビネット・オブ・キュリオシティーズ)」への批判、進化主義的な進歩史観から
文化相対主義へと至る理論的転換までを辿る。マリノフスキーによる参与観察の確立、ダーウィンの
先住民描写に見られる客観的観察の限界、機能主義から象徴・パフォーマンス人類学への発展を扱い、
普遍的人権と文化的慣行の緊張関係(女性性器切除など)にも言及し、他者理解には社会の政治的・
経済的文脈への共感が不可欠だと結論づける。
