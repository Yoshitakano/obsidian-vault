# ヤマギシを考える全国ネットワーク_ヤマギシズム学園の光と影

「ヤマギシズム学園」の理想化された公的イメージと教育実態の乖離を批判的に分析するテキスト。
幼少期から大学レベルまでの発達段階を通じ、絶対的な親子分離と過酷な共同労働が家族の絆の
断絶と個人のアイデンティティの喪失をもたらす仕組みを詳述する。「素直な子」の育成が実際には
組織的な感情抑圧、正規学業の欠如、個人の成長より集団への統合を優先する服従文化によって
達成されていると論じ、労働集約的な自給自足と「人間改造」への注力が家族の分断と組織的統制の
メカニズムとして機能すると警鐘を鳴らす。
