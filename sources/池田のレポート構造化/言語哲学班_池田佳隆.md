# 言語哲学班_池田佳隆

ウィトゲンシュタイン『論理哲学論考』が示す世界の構造を解説するプレゼンテーション。世界は物の
集合ではなく「事実」から成り立ち、事実は単純な「対象」が結合して成立する「事態」の実現である
と説明する。あらゆる可能性は無限の「論理空間」に含まれ、対象の「内的性質」に基づいてどう結合
できるかを規定する。対象の「形式」は論理的必然として一定である一方、その「内容」と関係性は
変化しうるとし、事実の成立・不成立を写し取る論理的写像として世界理解を捉える。
