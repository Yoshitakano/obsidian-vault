# 柳宗悦_手仕事の日本_プレゼン新

民藝運動の創始者・柳宗悦の思想を、健全で平和な日本社会の青写真として提示するプレゼンテーション。
著書『手仕事の日本』を通じ、真の美は個人のエゴではなく自然と伝統に根ざす無名の職人が作る
機能的で日常的な物にこそ宿ると論じる。これらの手仕事の品は「健康の美」という道徳的基盤を
体現し、正直で実用的な工芸に囲まれた国はより健全な公共生活を育めると示唆する。自国の文化的
ルーツを深く尊重することが他国への誠実で普遍的な敬意へとつながる「平和なナショナリズム」を
提唱し、工芸という共通言語を通じて世界の分断を架橋しようとする。
