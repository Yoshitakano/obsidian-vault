# 修正版_Group_05_JA05_日本語版アンケート

韓国における「サードプレイス」概念を調査するためにAPU学生が作成した研究用アンケート。回答者の
デモグラフィックデータを収集した後、家庭・職場・学校といった主な環境と、それ以外にリラックスや
自己表現のために利用する外部の場所への認識を比較する。利用頻度・アクセスのしやすさ・情緒的
便益に関する設問により、地域コミュニティへの帰属意識と個人のウェルビーイングを育む要因を
把握し、地域社会への関わりや国内外の様々な社会課題への関心度も評価する。
