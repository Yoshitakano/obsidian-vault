# コロナ前後での訪日外国人観光客数の国別変化

新型コロナウイルス流行前(2019年)と回復期(2023年)の訪日観光データを比較分析するレポート。
全体では約79%まで回復した一方、中国・ロシアからの大幅減少と、韓国・メキシコ・トルコなど
一部国からの成長という国別の顕著な差異を指摘。為替変動、地政学的対立、便数の増加などの要因を
挙げ、旅行者の嗜好が「モノ消費」から体験型観光へ移行していると分析し、地域体験を重視した
日本観光業の持続可能な発展への示唆を提供する。
