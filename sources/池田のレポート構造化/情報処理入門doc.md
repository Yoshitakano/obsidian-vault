# 情報処理入門doc

情報処理とデータ分析入門講義の教材を集約したアーカイブ文書。パンデミック前後の訪日観光動向の
変化に関する調査を中心に据え、共同のグループワーク資料と個人研究を通じて、国籍ごとの旅行
パターンの違いを検証する。データ整理技術を現実の経済・社会動向の分析に応用する実例を示す。
