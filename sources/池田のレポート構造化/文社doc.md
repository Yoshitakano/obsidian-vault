# 文社doc

社会科学(社会学・人類学)講義のための教材を集約したデジタルアーカイブ。講義資料・課題プロンプト・
音声記録など多様な媒体を通じ、一学期分の学習内容を時系列に整理する。社会契約論、文化相対主義、
宗教社会学といった主要テーマを扱い、マックス・ウェーバー、ルター、カルヴァンら思想家に焦点を
当てる。個人と集団という基礎概念から共同体と人間の死という複雑なテーマまでを導く体系的な
学習ロードマップとして機能する。
