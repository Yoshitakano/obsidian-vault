# 西洋哲学4リアペ＿宗教的人間にとっての科学

科学的客観性と技術進歩が支配する現代における「宗教的人間」の意義を論じるリアクションペーパー。
科学は「どのように」機能するかの機械論的理解に優れる一方、人間存在の背後にある主観的意味や
道徳的目的への対応には限界があると指摘。核エネルギーのような強力な技術をどう使うべきかという
倫理的判断には宗教が不可欠な指針を与えると論じ、精神的基盤としての宗教の意義を確認する。
