# JCクラス_2-C_研究計画発表

動物を伴う文化的祭事を現代においても存続させるべきかという倫理的問いを扱う研究計画発表。
感情的な動物擁護論と客観的な「動物福祉」基準を区別し、これらの伝統が地域アイデンティティと
社会的結束を育みつつ、経済的・生態学的利益も伴うと論じる。祭事の主目的が生存から観光・保全へと
シフトしても、実践方法を現代の倫理観に合わせて調整することで伝統は存続可能であり、維持されるべき
だと結論づけ、遺産が本質的な社会的価値を失わずに変化していくためのバランスの取れた枠組みを提示する。
