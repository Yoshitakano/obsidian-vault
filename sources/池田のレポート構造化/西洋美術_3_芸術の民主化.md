# 西洋美術_3_芸術の民主化

美術が支配階級の独占的な贅沢品から市民の日常生活を称える媒体へと変遷した歴史を辿るエッセイ。
貴族の距離感ある肖像画と、フェルメールらによる親密で身近な風俗画を対比し、芸術の
「民主化」を描き出す。真の芸術的価値は市場価格や名声によってではなく、人の心を動かす
普遍的な力によって定義されるべきだと主張し、芸術を日々の経験を豊かにする個人的な創造の道具
として受け入れることを促す。
