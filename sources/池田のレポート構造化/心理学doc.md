# 心理学doc

心理学入門講義の教材群を一つの体系的なカリキュラムとして整理したアーカイブ文書。性格検査や
自己認識といった理論的アセスメントと、児童虐待・社会的孤立などの重要な社会課題を橋渡しする
構成になっている。認知症ケア、環境要因、治療的介入といった心理学の実践的応用も扱い、
人間行動の複雑性と脆弱な人々を支える支援体制を理解するためのロードマップとして機能する。
