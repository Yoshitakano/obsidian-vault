# 加齢臭の誕生_調査研究法JB_期末レポート

資生堂が1999年に「加齢臭」を発見・命名して以降、この概念が日本でいかに社会的に構築されたかを
検証する調査研究法の期末レポート。新聞記事を分析し、明治期に公衆衛生上のリスクとされた臭いが、
現代では個人の道徳的義務・企業マーケティングの対象へと変化した歴史的過程を追う。曖昧だった
生理現象が、社会的排除や「スメハラ」というレッテルを避けるために厳格に管理すべき科学的
「事実」へと変貌した経緯を分析し、こうした衛生基準の進化がストレスといった目に見えない
心的状態までも「可視化」して自己管理させる文化を内面化させてきたと論じる。
