# 西洋哲学3リアペ＿人間化は自文化中心

"hominization"の訳語として偏った優越性を含む「人間化」ではなく中立的な生物学的訳語「人化」を
用いるべきだと論じるリアクションペーパー。人間を存在の頂点と見る視座は、他文化を貶めてきた
自文化中心主義と同型であり、現在は地球環境を脅かしていると指摘。人間中心主義的偏見を退け、
生物学的相対主義に立つことで、あらゆる生命の内在的価値を認め環境破壊を回避できると主張する。
