# 心理学_3_性格における自己認識と性格検査結果の異同

自己の内的認識と客観的な性格検査結果との食い違いを、共感性や論理性をめぐる筆者自身の矛盾を
事例として考察する。この乖離は、理想と現実を混同しがちで客観的な自己観察が難しいこと、また
標準化検査が状況依存的な行動のニュアンスを捉えきれないことから生じると論じる。個人の直感と
臨床的評価いずれも絶対的な真実とみなすべきではないと指摘し、両者を統合することで自分が
どうありたいかとの間の溝を埋める多面的な自己理解が可能になると結論づける。
