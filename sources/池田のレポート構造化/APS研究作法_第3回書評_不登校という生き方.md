# APS研究作法_第3回書評_不登校という生き方

『不登校という生き方：教育の多様化と子どもの権利』の書評。日本の不登校対応の歴史を「首縄」期
(懲罰的)、「軟化」期(共感的)、「揺り戻し」期(制限的)の三段階に区分し、社会的圧力が強制と
受容の間で揺れ動く様を描く。草の根団体やフリースクールが子どもの人権擁護に果たした役割を
重視し、個人を責めるのではなく学校システム自体が子どもの主体性と幸福を尊重するよう
再構築されるべきだと主張する。
