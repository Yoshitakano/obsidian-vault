# 調査計画シート_05

FIRSTプログラムの学生向けに、学術理論から実地調査への体系的な移行を詳述する調査計画ガイド。
前半ではグループプロジェクトのための六段階の手法(問いの設定基準、偏りのない調査設計、
倫理的なインタビュー実施)を扱う。後半ではこの原則を「サードプレイス」(家庭・職場以外の
心地よい社会的空間)が韓国地方都市のコミュニティ結束と個人の幸福に与える影響という具体的な
事例研究に適用し、日本の文脈と対比しながら、こうした社会的拠点がどのように現代の社会的孤立を
緩和し地域アイデンティティを再活性化させ得るかを、構造化されたデータ分析と発表を通じて
明らかにすることを目指す。
