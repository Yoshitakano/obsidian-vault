# Study_Plan_授業料減免

授業料減免奨学金申請のために池田佳隆が提出した学習計画書。日本の伝統的民藝(工芸)の担い手となり、
文化遺産を次世代に継承するという明確な将来のキャリア展望を示す。目標達成のため、社会学・人類学の
学際的な履修と批判的思考力・実践経験の育成を組み合わせた綿密な学業戦略を詳述し、将来のキャリアの
確かな基盤となる強い学業意欲と知的好奇心を示す。
