# レポート添削基準の確認

この文書は、APUライティングマニュアルVer.2.0に基づき、AIアシスタントが22名の学生レポートを評価する
ための厳格な枠組みを確立するものである。主な目的は、言語的な精密さ、標準化された体裁、適切な引用
実践という3つの柱に焦点を当て、学術的誠実性とプロフェッショナルな書式を確保することにある。具体的
には、客観的な「だ・である調」の文体、特定のフォント・余白設定、ハンギングインデント付きでアルファ
ベット順に整理された参考文献リストが求められる。AIは物理的な文書設定を視覚的に確認できないため、
テキスト上の誤りを指摘しつつ学生自身に視覚的な体裁を再確認するよう促す協働的なフィードバックツール
として機能する。最終的にこのやり取りは、採点基準を洗練し、全学生の評価における一貫性を維持するための
基幹参照スレッドとして機能している。
